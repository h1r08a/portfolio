
# 概要
### Web上で日記の作成・編集・削除ができるサービス
リンク：[https://naganaga10.tk/](https://naganaga10.tk/)
---
## 機能一覧
* 日記投稿
* 日記一覧表示
* 日記詳細表示
* 日記編集
* 日記削除
* 管理ユーザー登録
* 管理ユーザログイン
* 画像ファイルのアップロード
* ページネーション
---
## 技術/サービス
* 言語: Python3
* フレームワーク: Django
* データベース: PostgreSQL
* ソース管理: Git/Bitbucket
* CSS/JS: Bootstrap
* 承認機能: django-allauth
* 画像アップロード: Pillow
* インフラ: AWS_EC2 / Amazon Linux2
* Webサーバー: Nginx
* APサーバー: Gunicorn

* Library: [requirements.txt](#)
